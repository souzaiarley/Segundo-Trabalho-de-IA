Instruções de Execução
------------------------------------------------------------

Execução Principal (main.py):
Para rodar o algoritmo genético conforme desejado, basta executar o seguinte comando no terminal:

    python main.py

O programa guiará você pelo fluxo de execução necessário.

Caso queira utilizar instâncias personalizadas do Problema Quadrático de Alocação (PQA), basta preencher o arquivo "customInstance.txt" com:

1. Coordenadas dos locais
2. Matriz de fluxo das instalações

------------------------------------------------------------

Execução dos Experimentos:
Os arquivos de experimentos estão organizados na pasta "experiments", cada um dentro de sua própria subpasta.

Para executá-los, utilize o comando "python". Por exemplo, para rodar o experimento 1, basta executar:

    python experiment_1.py

Análise dos Resultados:
Os resultados dos experimentos são armazenados em arquivos ".csv", gerados ao final da execução. Nesses arquivos, você encontrará informações detalhadas, como:

- Dados sobre as gerações
- Instâncias do problema

Essas informações podem ser úteis para avaliar o desempenho dos algoritmos.
